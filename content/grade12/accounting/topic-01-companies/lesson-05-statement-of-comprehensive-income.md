# Statement of Comprehensive Income

Also known as the **Income Statement**.
**Goal:** Calculate the Net Profit or Loss for the financial year.

## Standard Format

| Account | Details |
| :--- | :--- |
| **Sales** | |
| Cost of Sales | *(Less)* |
| **Gross Profit** | **(A)** |
| Other Operating Income | (Add ALL income) |
| -- *Rent Income* | *(Adjust for prepaid/accrued)* |
| -- *Commission Income* | |
| **Gross Operating Income** | **(B)** |
| Operating Expenses | (Less ALL expenses) |
| -- *Salaries & Wages* | |
| -- *Depreciation* | *(Often needs calculation)* |
| -- *Audit Fees* | |
| -- *Directors Fees* | |
| -- *Sundry Expenses* | |
| **Operating Profit** | **(C)** |
| Interest Income | (Add) |
| **Profit Before Interest Expense** | **(D)** |
| Interest Expense | (Less) |
| **Net Profit Before Tax** | **(E)** |
| Income Tax | (Less) |
| **Net Profit After Tax** | **(F)** |

---

## Crucial Adjustments (Year-End)

In Grade 12, you are never just copying figures. You must adjust for:

### 1. Depreciation

- **Cost Price Method**: `Cost × %`
- **Diminishing Balance Method**: `(Cost - Accumulated Depreciation) × %`
- Remember to apportion for time if an asset was bought/sold during the year.

### 2. Loan Interest

- Often capitalized (added to loan).
- If the Trial Balance shows "Loan Interest (paid)", check if the total matches the calculated interest. If not, the difference is accrued interest (add to expense).

### 3. Directors Fees

- Fees paid to directors are an **Operating Expense**.
- Do not confuse with dividends (profit share).

### 4. Audit Fees

- Only the **Audit Fees** expense goes here.
- Do not include "Audit fees paid" if it settles last year's debt. Adjust for amounts owing.

---

## Exam Tips

- **Show workings in brackets!** e.g., Salaries `(150 000 + 12 000 accrued)`. You get part-marks for the logic even if the final answer is wrong.
- **Operating Profit**: Be careful. Interest Income is ADDED *after* Operating Profit. Interest Expense is DEDUCTED *after* that.
  - *Operating Profit = EBIT (Earnings Before Interest and Tax) excluding Interest Income usually for CAPS formatting strictness, but Interest Income is technically operational result. Follow strict CAPS format: Operating Profit -> + Interest Income -> - Interest Exp.*
- **Income Tax**: If not given, calculate it as % of Net Profit Before Tax.

---

## Common Mistakes

- Putting Interest Expense inside Operating Expenses. (It must be separate).
- Forgetting to adjust Rent Income for amounts received in advance (subtract).
- Calculating depreciation on the wrong amount (Cost vs Carrying Value).
