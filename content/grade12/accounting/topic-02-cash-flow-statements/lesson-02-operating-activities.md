# Operating Activities

This is usually the biggest section and involves a detailed note: **Reconciliation of Profit before Tax and Cash Generated from Operations**.

## The Reconciliation Note (Note 1)

We start with **Profit Before Tax** and reverse all non-cash items and non-operating items.

**Format:**

| Details | R |
| :--- | :--- |
| **Profit Before Taxation** | **X,XXX** |
| Adjustments for: | |
| -- Depreciation | + XX |
| -- Interest Expense | + XX |
| **Operating Profit before changes in Working Capital** | **X,XXX** |
| Changes in Working Capital | **X,XXX** |
| -- Change in Inventory | (Increase) / Decrease |
| -- Change in Receivables (Debtors) | (Increase) / Decrease |
| -- Change in Payables (Creditors) | Increase / (Decrease) |
| **Cash Generated from Operations** | **X,XXX** |

### Why do we add back Interest?

Because Interest Paid is shown separately on the face of the Cash Flow Statement. We remove it from the note to avoid double counting.

### Working Capital Logic

- **Increase in Asset (Inventory/Debtors)** = **Outflow** (We bought stock or haven't collected cash).
- **Decrease in Asset** = **Inflow** (We sold stock or collected cash).
- **Increase in Liability (Creditors)** = **Inflow** (We kept the cash instead of paying).
- **Decrease in Liability** = **Outflow** (We used cash to pay debts).

## Face of the Statement: Operating Section

Once Note 1 is done, we move to the main statement:

| Details | R |
| :--- | :--- |
| **Cash effects of operating activities** | |
| Cash generated from operations (From Note 1) | X,XXX |
| Interest Income | + XX |
| Dividends Received | + XX |
| Interest Paid | (XX) |
| Taxation Paid | (XX) |
| Dividends Paid | (XX) |

### Key Calculations

1. **Taxation Paid**:
    - Opening Balance (SARS)
    - Plus: Tax Expense (Income Statement)
    - Less: Closing Balance (SARS)
    - *= Amount Paid*

2. **Dividends Paid**:
    - Opening Balance (Shareholders for Dividends)
    - Plus: Dividends Declared (Total for year)
    - Less: Closing Balance (Shareholders for Dividends)
    - *= Amount Paid*

---

## Exam Tips

- **Depreciation** is *always added back* because it reduced profit but didn't use cash.
- **Interest Expense** is *added back* in the Note, then *subtracted* on the face (Interest Paid).
- **Tax Paid** and **Dividends Paid** are almost always outflows, so use brackets `( )`.
