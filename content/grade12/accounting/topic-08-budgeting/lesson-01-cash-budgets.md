# Cash Budgets

**Purpose**: To predict the future cash position (Liquidity).
**Remember**: Only **CASH** flows go here. No depreciation, no bad debts.

## Format

| Receipts | Jan | Feb |
| :--- | :--- | :--- |
| Cash Sales | XX | XX |
| Collection for Debtors | XX | XX |
| Rent Income | XX | XX |
| **Total Receipts** | **A** | **A** |

| Payments | Jan | Feb |
| :--- | :--- | :--- |
| Cash Purchases | XX | XX |
| Payment to Creditors | XX | XX |
| Wages | XX | XX |
| **Total Payments** | **B** | **B** |

| **Surplus / Deficit** (A - B) | XX | XX |
| Balance at beginning | XX | XX |
| **Balance at end** | **XX** | **XX** |

---

## Exam Tips

- **Wages vs Salaries**: Often paid in the same month.
- **Donation**: If we donate stock, it's NOT a cash payment. If we donate money, it IS a cash payment.
- **Depreciation**: NEVER put depreciation in a Cash Budget. It is a non-cash expense.

---

## Quick Check

1. Does "Bad Debts" appear in the Cash Budget? (No).
2. If we buy a vehicle on credit, does it go in the Cash Budget? (No. Only the monthly installlments paid).
3. Calculation: Surplus R10k, Opening Balance (R2k) overdraft. Closing Balance? (R8k).
